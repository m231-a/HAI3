## Submit Your Case Study

Have you built something with HAI3? We'd love to feature your work!

**What we're looking for:**
- Real production deployments
- Interesting architectural challenges
- Novel uses of HAI3 features
- Lessons learned and best practices

**How to submit:**
1. Create an issue on [GitHub](https://github.com/HAI3org/HAI3/issues) with the label "case-study"
2. Include a brief description of your project
3. Share any public code, blog posts, or documentation
4. We'll work with you to create a case study page
